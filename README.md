# 🏠 Smart Home GUI

Современный графический интерфейс для системы умного дома, разработанный на Python с использованием CustomTkinter.

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![Python](https://img.shields.io/badge/python-3.7%2B-green)
![CustomTkinter](https://img.shields.io/badge/CustomTkinter-5.2.0-orange)

## 📋 О проекте

Smart Home GUI - это современное приложение для управления умным домом с интуитивно понятным интерфейсом. Приложение позволяет контролировать освещение, климат, безопасность и энергопотребление в реальном времени.

### ✨ Основные возможности

- **💡 Управление освещением** - контроль света в разных комнатах
- **🌡️ Климат-контроль** - мониторинг и регулировка температуры и влажности
- **🔒 Система безопасности** - контроль доступа и состояние датчиков
- **📊 Мониторинг энергии** - отслеживание энергопотребления
- **🎬 Готовые сцены** - быстрые сценарии для разных ситуаций
- **🌓 Темная тема** - современный дизайн с темной темой

## 🚀 Установка

### Предварительные требования

- Python 3.7 или выше
- pip (менеджер пакетов Python)

### Пошаговая установка

1. **Клонируйте репозиторий**
```bash
git clone https://github.com/your-username/smart-home-gui.git
cd smart-home-gui
Создайте виртуальное окружение (рекомендуется)

bash
python -m venv venv

# Активация на Windows
venv\Scripts\activate

# Активация на macOS/Linux
source venv/bin/activate
Установите зависимости

bash
pip install -r requirements.txt
Или установите вручную:

bash
pip install customtkinter Pillow
Запустите приложение

bash
python script.py

Настройка цветовой схемы
В файле script.py вы можете изменить цветовую схему:

python
self.colors = {

    'danger': "#e74c3c",        # Цвет опасности
    'surface': "#242424",       # Цвет поверхностей
    'text': "#ffffff"           # Цвет текста
}
